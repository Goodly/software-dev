To view the sunburst.html, do the following:

Open your command line and run:
npx http-server

This will start a local server at localhost:8080
Go to localhost:8080 and navigate to the following directory: software-dev/Visualizations/rebuild

Click sunburst.html to view the sunburst.
Hovering over the arc will tell you the point value and the name of the data associated with the arc.
